Перші програми

Python — це одна з найпопулярніших і найпростіших мов програмування. Її використовують для створення ігор, сайтів, додатків і навіть штучного інтелекту. Щоб почати, потрібно встановити Python і відкрити програму IDLE, у якій ми пишемо код. Перша команда, яку вчать усі, — print(). Вона показує на екрані те, що ми записали в дужках, наприклад: print("Привіт, світе!"). У Python є змінні — це місце, де зберігається інформація.

Наприклад, name = "Олег" зберігає ім’я, а print(name) виведе його на екран. Є різні типи даних: числа (int, float), текст (str) і логічні значення True або False. Python може рахувати (+, -, *, /) і “склеювати” текст, як у прикладі: print("Привіт, " + name + "!"). Важливо писати уважно: не забувати лапки, дужки й відступи. З часом ви навчитеся створювати умови (щоб програма приймала рішення) і цикли (щоб повторювати дії). Python — це легкий і цікавий спосіб почати програмувати. Головне — не боятися пробувати і вчитись на помилках!
